### Project Files Overview
1. **main.c (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c)**:
   - Implements the C program.
   - Defines a `char canvas[HEIGHT][WIDTH]` to store the picture (where background characters are `_` and borders are `*`).
   - Standard drawing algorithms:
     - draw_line (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c#L101-L125) (Bresenham's line algorithm)
     - draw_circle (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c#L143-L162) (Bresenham's Midpoint Circle algorithm)
     - draw_rectangle (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c#L165-L179) (draws top, bottom, left, and right borders)
     - draw_triangle (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c#L182-L186) (draws three connected lines between coordinates)
     - display_canvas (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c#L211-L229) (draws upper/lower and side borders around the picture and outputs the 2D grid)
   - Features robust input validation using get_int_input (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/main.c#L55-L71) to ensure menu and coordinate selections do not lead to infinite loops.
   - Preserves shapes dynamically in a list so that you can easily list, modify, or delete them before re-rendering the output.
   - Pre-loads a demo scene (a house base, triangle roof, and circular sun) upon startup.
2. **Web GUI Suite**:

### Compilation and Execution instructions
#### C Application:
Compile it using your terminal's C compiler (like GCC or Clang):
```bash
gcc -o editor main.c -Wall
```
To run the editor:
- **Windows**: `.\editor.exe`
- **Linux/macOS**: `./editor`
#### Web GUI Application:
Simply open **index.html (file:///C:/Users/deety/.gemini/antigravity/scratch/2d-ascii-graphics-editor/index.html)** in any modern web browser (Double click or drag it into Chrome/Edge/Firefox) to design shapes interactively!
